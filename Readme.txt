Media Player v1.0

Build instructions:

1. Open project in Visual Studio 2015 or Visual Studio 2019. 
2. NET Framework version does not matter. Select any if it will requires.
3. Go to main menu, select Build, Buid solution.

Note:
If you want to have Media Player component within your toolbox than add it.

Right click on toolbox, select Choose Items... menu. Go to COM components, check Windows Media Player.
